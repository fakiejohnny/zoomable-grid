# zoomable-grid
smoothly zoomable Grid layout

### [Demo](https://fakiejohnny.github.io/zoomable-grid/)

## controls
* mouse wheel
* mouse click
* some keys (arrow keys, escape, +, -, 0, ...)
* find out for yourself - it's easy to understand :)

## technique
* CSS3 transform(translate, scale)
* pure JavaScript
* **-> no dependencies**
